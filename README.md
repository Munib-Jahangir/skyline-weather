## ğŸš€ Getting Started

### Prerequisites
- Node.js (Latest LTS)
- Android SDK (for mobile builds)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Munib-Jahangir/skyline-weather.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally
```bash
npm run dev
```

### Building for Android
```bash
npm run build
npx cap sync
cd android
./gradlew assembleDebug
```
