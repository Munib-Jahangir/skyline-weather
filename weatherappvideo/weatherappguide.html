Capacitor Mobile Setup Guide

0. Essential Prerequisites

Node.js
Latest LTS Version (v20+)
Java JDK
OpenJDK 21 (v21.0.8+)
Android Studio
Latest with Android SDK
1

Install Core Tools

Ensure you have Java 21 (OpenJDK) and Android Studio installed.

npm install @capacitor/core @capacitor/cli @capacitor/android
2

Initialize & Add Android

Create the config file and add the native folder.

npx cap init "Skyline Weather" "com.munib.skylineweather" --web-dir dist npx cap add android
3

Assets (Icons & Splash)

Auto-generate all screen sizes using a single 1024x1024 logo.

npm install @capacitor/assets --save-dev npx capacitor-assets generate --android
4

Build & Sync

Build your web project first, then sync to Android folder.

npm run build npx cap sync
5

Generate APK

Run via terminal or open Android Studio.

cd android; ./gradlew assembleDebug
